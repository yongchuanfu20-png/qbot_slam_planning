# qbot_slam_planning_python
Developed in Quanser platform, with function of localization, mapping, path planning, following etc.

I. To get start, Please download GitHub resources: https://github.com/quanser/Quanser_Academic_Resources.git
with git: git clone https://github.com/quanser/Quanser_Academic_Resources.git Quanser
**MAKE SURE IT IS IN YOUR DOCUMENTS FILES: C:/Users/user/Documents


II. go to docs\pc_setup.md, following "if you are using Python" instruction:
1. update python
2. download Quanser SDK
3. download Quanser interactive Labs
4. restart your computer

III. go to 1_setup folder
1. run step_1_check_requirements.bat and select:
	python only
	both virtual and hardware
   to check the set up files
2. run configure_python

IV. Open the files 2_quick_start_guides\python follow the "quick_start_guide_virtual_python.pdf" to get start in virtual env.

V.  After be familar with this platform, run observer and obstacle_detection under 'auto_driving_car' file with Quanser interactive Labs
